# Privacy Policy – .mcas.ms Auto-remover

Last updated: 9 March 2026

## Overview

.mcas.ms Auto-remover is a browser extension designed to correct certain Microsoft Defender proxy URLs by removing the ".mcas.ms" suffix from addresses containing "/v5/ui/". This allows affected web pages to function correctly when uploading or downloading documents.

The extension performs this correction locally within the user's browser.

## Data Collection

This extension does **not collect, store, transmit, or share any personal data**.

The extension does not collect:

* Personal information
* Browsing history
* Website content
* Authentication information
* Files or documents
* Analytics or telemetry data

## How the Extension Works

When the browser attempts to load a webpage with a URL matching:

```
*.mcas.ms/v5/ui/*
```

the extension automatically removes the ".mcas.ms" portion of the hostname and redirects the browser to the corrected address.

This process occurs entirely within the user's browser and does not involve any external servers or third-party services.

## Permissions

The extension requests access to URLs matching:

```
*.mcas.ms/v5/ui/*
```

This permission is required solely to detect and redirect affected URLs.

The extension does **not read or modify page content**, and it only operates on the specific URL pattern described above.

## Third-Party Services

This extension does not use:

* external APIs
* analytics services
* tracking tools
* advertising networks

## Data Storage

The extension does not store any data locally or remotely.

## Changes to This Policy

If the functionality of the extension changes in the future, this privacy policy may be updated accordingly.
